命令行下的光标快捷键

其实有很多命令，但是这里只列举出一些常用的

// 光标移动
ctrl + e // 移动到行尾
ctrl + a // 移动到行首
alt  + b // 移动到光标处单词尾
alt  + f // 移动到光标处单词首
// 复制，剪切和粘贴
ctrl + u // 剪切光标前的所有内容
ctrl + k // 剪切光标后的所有内容
ctrl + y // 粘贴内容
