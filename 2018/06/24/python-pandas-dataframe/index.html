pandas(Python Data Analysis Library)数据结构 —— DataFrame

1. 什么是pandas

pandas是一款基于NumPy的数据分析库。
pandas里有两种自己独有的基本数据类型：Series和DataFrame。
有了这两种数据类型，数据分析便变得简单。

前期别忘了导入它们：

import pandas as pd

2. DataFrame的创建，保存和读取

2.1 创建DataFrame

print("---------- write teset ----------")
df2 = pd.DataFrame({"number":[1,2,3],
                    "name":["lily", "lucy", "jacky"],
                    "age":[16,18,22]},
                    columns=["number", "age", "name"]) #用于决定列的排列顺序

print(df2)

运行结果如下所示：

---------- write test ----------
   number  age   name
0       1   16   lily
1       2   18   lucy
2       3   22  jacky

第一列的0 1 2表示索引值。

2.2 将DataFrame保存为csv文件

df2.to_csv("test1.csv", index=0, se=',') # index=0表示不保存索引行

2.3 读取csv文件

df1 = pd.DataFrame(pd.read_csv('name.csv',header=1))
print(df1)

输出结果如下，可以看出跟我们输入的完全一样。

---------- read test ----------
   number  age   name
0       1   16   lily
1       2   18   lucy
2       3   22  jacky

3. 获取DataFrame的基本属性

承接上一章节中创建的DataFrame。

3.1 查看DataFrame的结构/形状

print(df2.shape)

数据结构为一个3X3矩阵。

3.2 查看各列的数据类型

print(df2.dtypes)

运行结果：

4 数据的提取和筛选

4.1 以矩阵形式打印出所有/某列/某行数据

print("------- 打印全部数据 ---------")
print(df2.values)        # 打印全部数据
print("------- 打印第一行数据 -------")
print(df2.values[0])    # 打印第一行数据
print("------- 打印第一列数据 -------")
print(df2.values[:,0])    # 打印第一列数据

运行结果如下：

------- 打印全部数据 ---------
[[1 16 'lily']
 [2 18 'lucy']
 [3 22 'jacky']]
------- 打印第一行数据 -------
[1 16 'lily']
------- 打印第一列数据 -------
[1 2 3]

4.2 获取index(索引)值和column(列)名称

print("------------------ print index -----------------")
print(df2.index) # 获取index(索引)值
print("----------- print the list type index ----------")
print(list(df2.index))
print("------------------ print column ----------------")
print(df2.columns) # 获column(列)名称
print("----------- print the list type column ---------")
print(list(df2.columns))

运行结果：

------------------ print index -----------------
RangeIndex(start=0, stop=3, step=1)
----------- print the list type index ----------
[0, 1, 2]
------------------ print column ----------------
Index(['number', 'age', 'name'], dtype='object')
----------- print the list type column ---------
['number', 'age', 'name']

4.3 根据column name查看数据

e.g. 查看age列的数据：

print(df2['age'])

运行结果：

Name: age, dtype: int64

4.4 根据index查看某个成员

print(df2.loc[2])

现在我们把样本扩大，从之前的3个扩到到8个：

“age”:[16,18,22,5,10,40,66,30],
“sex”:[“female”,”female”,”man”,”man”,”female”,”man”,”man”,”man”]},
columns=[“number”, “age”, “name”, “sex”]) #用于决定列的排列顺序

print(df2)
print(“———- end —————–”)

4.5.1 数据筛选之”与或非”的使用

（1）提取age大于等于18的数据(‘age’>=18)
（2）提取age大于等于18(‘age’>=18)且age小于40的数据
（3）提取age大于等于50(‘age’>=18)或age小于10的数据
（4）提取sex为非man的数据

print("---------- age >= 18 ----------------------")
print(df2.loc[df2['age']>=18])
print("---------- age >= 18 且 age < 40 ----------")
print(df2.loc[ (df2['age']>=18) & (df2['age']<40) ])
print("---------- age >= 50 或 age < 10 ----------")
print(df2.loc[ (df2['age']>=50) | (df2['age']<10) ])
print("---------- sex != man ---------------------")
print(df2.loc[ (df2["sex"] != "man")])

运行结果：

4.5.2 数据筛选并计数

print("---------- age >= 18 ----------------------")

print(df2.loc[df2['age']>=18]['age'].count())
