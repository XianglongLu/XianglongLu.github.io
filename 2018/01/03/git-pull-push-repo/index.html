Git —— 将本地修改同步到远程仓库 & 将本地仓库与远程仓库同步

前言 / Preface

有时候想删除仓库中的一些文件或文件夹，具体该如何操作呢？方法如下。

正文 / Body

配置你的git账号和邮箱并初始化：

git config --global user.name "MyUserName"
git config --global user.email "xxxxxxx@yy.com"
git init

把远程仓库clone到本地

$ git clone https://git.coding.net/xxx/xxx.git

1 将本地修改同步到远程仓库

假设你修改了文档xxx.txt或者新建了xxx.txt（假设它在yyy目录下面），想要将本地的这个修改同步到远程仓库。

Step 1 添加修改文件

$ git add yyy/xxx.txt

Step 2 commit并添加注释

$ git commit -m "add/modify xxx.txt"

Step 3 推送到远程仓库

$ git push origin branch_name

2 将本地仓库与远程仓库同步

假设远程仓库发生了改变，但你想让本地仓库与远程仓库同步。

$ git pull

则可完成同步。
