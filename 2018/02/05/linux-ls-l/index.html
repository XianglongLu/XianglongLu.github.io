Linux —— ls -l 命令解析

正文

ls -l的作用：是以长格式形式显示当前目录下所有可见文件的详细属性。

当在linux终端下敲下

$ ls -l

后会罗列出类似于如下的几行信息。

1. 文件类型解析

d : 目录
- : 文件
l : 链接
s : socket
p : named pipe 
b : block device
c : character device

2. 权限解析

如上图所示，权限共有9个字符组成，其中每3个作为一组。
第一组代表文件所有者的权限。
第二组代表文件所在用户组的权限。
第三组代表一般用户所拥有的权限。

首先你需要了解以下符号所代表的的信息：

r // 读权限
w // 写权限
x // 执行权限
- // 无权限
rwx // 拥有读写和执行权限
rw- // 拥有读写，但没有执行权限

如果你想要修改权限，需要用到chmod指令。

$ chmod 765 xx.py // 765 = rwxrw-r-x
$ chmod 432 xx.py // 432 = r--r-x-w-
$ chmod 100 xx.py // 100 = --x------
