长命令行选项解析函数getopt_long()

1. 前言

getopt()比较常见，用于解析命令行短选项的一个函数。
定义：

int getopt(int argc, char * const argv[], const char *optstring);

它可以解析短选项，如：

-d 5 -c 2

但是它却没法解析长选项，如：

--device 5 --client 2

getopt_long()则不一样，它不仅包含getopt()的功能，还可以用于解析长选项问题。
定义：

int getopt_long(int argc, char * const argv[], const char *optstring,
                const struct option *longopts, int *longindex);

与getopt对比可知，其实getopt_long多了2个参数，即const struct option longopts, int longindex。

调用getopt_long()前你需要定义选项结构体，其定义如下：

struct option {
const char    *name;        // 参数名称
int            has_arg;    // 指明是否带有参数，0代表不带参数，1代表带参数，2代表参数可选
int            *flag;        // flag=NULL时，返回value；不为空时，*flag=val，返回0
int            val;        // 用于指定函数找到选项的返回值或flag非空时指定*flag的值
};

示例：

struct option long_options[] = 
    {
        {"name",    1,    NULL,    'n'}, // 参数名称为name，必须带参数，返回value（即'n'），检测到选项name时返回'n'
        {"old",        2,    NULL,    'o'}, // 参数名称为old，带不带参数都行，返回value（即'o'），检测到选项old时返回'n'
        {"end",        0,    NULL,    'e'}, // 参数名称为end，不带参数，返回value（即'e'），检测到选项end时返回'e'
        {0,            0,    0,        0},
    };

值得一提的时，当has_arg=1时（带参数），你可以如下2种方式使用：

--name 10 // 方法1，空格隔开参数
--name=10 // 方法2，等号后面跟着参数

但当has_arg=2时（带不带参数都行），如果你要带参数，你只能够使用上述的第2种方法：

--old=5

2. 例程分析

写一个叫做test的命令，它拥有3个长选项，分别是name、old和end。
其中name必须带参数，old带不带参数为可选的，end不带参数。

#include<stdio.h>
#include<getopt.h> // 必须包含该库才可以使用命令行选项解析函数哦~

int usage(void)
{
    printf("USAGE:\n./test --name=[name] --old=[old] --end\n");
    return 0;
};

char * short_options = "n:oe"; //带:的意思是n接受参数
struct option long_options[] = 
{
    {"name",    1,    NULL,    'n'},
    {"old",        2,    NULL,    'o'},
    {"end",        0,    NULL,    'e'},
    {0,            0,    0,        0},
};

int main(int argc, char *argv[])
{
    int c;

    if(argc == 1) // 如果test命令不带任何选项
    {
        usage();
        return -1;
    }

    while( ( c = getopt_long(argc, argv, short_options, long_options, NULL) ) != -1 )
    {
        switch(c)
        {
            case 'n':
                printf("My name is %s.\n", optarg);
                break;
            case 'o':
                if ( optarg == '\0' )
                    printf("My age is a secret.\n");
                else
                    printf("I am %s.\n", optarg);
                break;
            case 'e':
                printf("The end.\n");
                break;
            default:
                usage();
                break;
        }
    }
    return 0;
}

参数使用测试：

// 注意，--old带参数时候必须加"="等号
$test --name=jim --old=18 --end
My name is jim.
I am 18.
The end.

$test --name=jim --old --end
My name is jim.
My age is a secret.
The end.

// 如果该带参数的你忘记带了，则会如下
$test --name --old
My name is --old.

注意！虽然getopt_long包含了getopt的功能。但是如果你的参数包含了带不带参数都行的选项，如上述的–old。
你最好还是按照长选项来使用。
因为在short_options的定义中，只有2种情况，即带参数和不带参数。
根据上述定义，即”n:oe”，无论你在应用中-o带不带参数，都按照不带参数处理。

$test -n jim -o -e
My name is jim.
My age is a secret.
The end.

$test -n jim -o 18 -e
My name is jim.
My age is a secret.
The end.
