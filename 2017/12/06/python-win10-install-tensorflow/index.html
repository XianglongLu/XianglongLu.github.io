【教程】Win10环境下安装TensorFlow

前言 / Preface

谷歌用5个词很好的概括了什么是机器学习：using data to answer question.
对于tensorflow，我相信已经无需多言，就直接进入正题吧。

正文 / Body

Step 1. 安装python环境（建议安装Anaconda）

Anaconda除了集成python的环境，还带有很多安装tensorflow所必须的软件库。
直接到官网下载安装Anaconda即可，非常方便。
https://www.anaconda.com/download/
我安装的是Anaconda2。

Step 2. 安装python环境（建议安装Anaconda）

启动Anaconda prompt，创建一个名为tf_py35（名字可自定义）的python 3.5的环境，指令如下：
名字可自定义。

$ conda create --name tf_py35 python=3.5

注意：如果下载速度过慢，可以尝试使用直接下载tensorflow的python安装包之后在安装。
激活tf_py35环境并安装tensorflow（我安装的是GPU版本）：

$ activate tf_py35
$ conda install tensorflow-gpu

注意：如果电脑不是Nvidia的独立显卡，那只能乖乖安装CPU版本了。

$ activate tf_py35
$ conda install tensorflow

Step 3. 测试安装是否成功

$ import tensorflow

没有提示错误则安装成功。
如果在spyder中使用tensorflow可能会提示没有tensorflow。
你需要的是在你的tf环境中安装spyder，启动Anaconda prompt，然后：

$ activate tf_py35
$ conda install spyder

之后你会发现电脑开始菜单的Anaconda目录下多了一个tf_py35的spyder。
使用该spyder，你可以发现import tensorflow成功了。
