Linux —— 强大的文本搜索工具grep

grep全称是Global search regular expression(RE) and Print out the line，全面搜索正则表达式并把行打印出来。
它是一种强大的文本搜索工具，它能使用正则表达式搜索文本，并把匹配的行打印出。

直接切入正题，常用例子供上~

多个文件中搜索某个字段

在xx.c，yy.c和zz.c里面寻找int字段：

grep "int" xx.c yy.c zz.c

在指定文件类型中搜索结果

e.g. 在当前目录的所以c文件中寻找abc字段：

grep "abc" -r --include="*.c" ./

其中

grep -n 显示行号
grep -i 不区分大小写
grep -E "[1-9]+"

显示匹配内容的前后几行信息

grep -A 3 可显示匹配内容及其后3行内容
grep -B 4 可显示匹配内容及其前4行内容
grep -C 5 可显示匹配内容及其前和后5行内容

过滤关键字

grep -v xxx test.c 过滤掉test.c中xxx
grep -v 'xxx\|yyy' test.c 过滤掉test.c中xxx和yyy
