Linux —— alsa-utils工具包之aplay, arecord和amixer简介

前言

  • 什么是ALSA？

ALSA是Advanced Linux Sound Architecture的缩写，高级Linux声音架构的简称,它在Linux操作系统上提供了音频和MIDI（Musical Instrument Digital Interface，音乐设备数字化接口）的支持。

ALSA在Linux系统中的架构大致如图所示。

即我们可以看到它主要由如下3部分组成：

  • alsa-utils
  • alsa-lib

  • alsa-driver
    此次我主要介绍其中的alsa-utils部分。

    正文

alsa-utils工具包中包含了如下3个命令行下的常用指令：

aplay - 用于播放音频
arecord - 用于录制音频
amixer - 用于配置和查看声卡的配置信息

其实它们也有图形界面版本，但本次主要介绍命令行版本。

1 aplay

aplay –l        // 列出系统当前有几个声卡和每个声卡下面挂着哪些设备

2 arecord

-r 采样率
-d 录制时间
-c channels (-c 2 代表立体音)

3 amixer

usage： amixer [command]

amixer –h

通过amixer contents可以查询出可控制的项目：

-c // 跟aplay和arecord不同，这是是声卡选项
-D // device
-d // debug mode
-v // 打印程序的版本信息

假设，我要设置sound card 1下麦克风录音时的音量大小(后面数字代表设置的音量大小)：

amixer -c 1 controls
// 查看card 1下Mic capture volume的numid


amixer -c 1 cget numid=11 // 查看当前音量大小


amixer -c 1 cset numid=11 31 // 将音量设置为31
