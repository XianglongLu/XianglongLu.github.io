Linux下的Coder利器 —— vi/vim之常用快捷命令

前言 / Preface

vi/vim（vi的升级版）是Linux下一款强大的IDE。
如果能够熟悉vim下的常用快捷指令，则可在coding中游刃有余。

正文 / Body

1. 进入和退出vim模式

vim有2个模式：（1）编辑模式（2）命令模式

  • 进入vim命令模式
    $ vim
  • 进入vim编辑模式
    直接进入vim编辑模式对文件main.c进行修改：
    $ vim main.c

  • 从vim编辑模式退出到vim命令模式
    在vim编辑模式中按下ESC按键，即可回到vim命令模式

  • 退出vim模式
    有人统计，Stack Overflow中有百万程序员在求助如何退出Vim。
    说句实话，如果你是个Linux初学者或者对vim不熟悉，那么这个问题确实会困扰你。

    :q     //不保存退出
:q!    //强制不保存退出
:wq    //保存退出
:wq!   //强制保存退出
:e!    //放弃所有修改并打开原文当

2. 快速跳转指令

gg      //跳至文章开头
G       //跳至文章结尾
:n      //跳至第n行
shift+b //跳至单词尾字母
shift+f //跳至单词首字母
ctrl+f  //向下翻页一屏
ctrl+b  //向上翻页一屏

3. 搜索指令

/xxx    //从光标处向后搜索xxx
?xxx    //从光标处先前搜索xxx
n       //跳到下一个搜索对象
N       //跳到上一个搜索对象

4. 复制粘贴删除撤销指令

yy      //复制当前光标所在行
nyy     //从光标所在行开始复制n行
pp      //在光标所在行下面粘贴
dd      //删除光标所在行
ndd     //类似于nyy
u      //撤销指令
ctrl+r  //恢复指令

5. 编辑指令

i       //在当前位置前插入
I       //在当前行首插入
a       //在当前位置后插入
A       //在当前行尾插入

6. 执行shell命令

:!xxx_shell_command
:!ls    //显示出当前文件所在目录的所有文件
:!pwd   //显示当前文件所在路径
