Linux —— (撰写中)MIPI CSI

basic info

MIPI Alliance: Mobile Industry Processor Interface Alliance 移动产业处理器接口联盟
CSI: Camera Serial Interface摄像机串行接口
Data lane: 数据通道
MIPI CSI-1，CSI-2和CSI-3是MIPI联盟Camera Working Group提出摄像机接口标准。

正文

MIPI CSI-2规范目前已经广泛用于移动产品中，它能很好的处理1080P，4K，8K甚至更高解析度拍摄得到的视频。

D-PHY在CSI-2使用时作为一个单向的差分接口，

C-PHY由1个或者多个单向3线串行数据通道组成，可成为trios（三人小组）。每个trios都有自己独立的嵌入式时钟。

C/D-PHY的CSI-2图像接口则没有限制每个链接的lanes数量。
