Linux —— apt包管理工具的常用指令

前言 / Preface

APT(Advanced Packaging Tool)是Linux命令行下的一个包管理工具。
常用的apt工具有：

apt-get
apt-cache
apt-file

此次主要介绍三者的一些常用指令。

正文 / Body

1. apt-get

更新软件安装源列表：

$ apt-get update

更新已安装软件到最新版本：

apt-get upgrade

安装一个新的软件包：

$ apt-get install package-name

安装指定版本的软件包：

$ apt-get install package-name=xx.x.yy.zz

卸载一个已安装的软件包：

$ apt-get remove package-name

2. apt-cache

搜索名字中包含python的软件包：

$ apt-cache search python

显示某个软件包的基本信息：

$ apt-cache show python

列出软件包的依赖：

$ apt-cache depends python

列出软件包的有哪些可选版本：

$ apt-cache madison python

3. apt-file

定期更新apt-file

apt-file update

查找某个.so文件所在的deb包：
e.g. libmpc.so.2

apt-file search libmpc.so.2
