Python —— Python的七大基本数据类型

前言

虽然早在2013年就开始使用Python做数据分析和图形绘制，但却一直没能进行系统的学习。悔不当初呀，现在就只能在上下班的公车上利用碎片化的时间进行自习了。

进入正题，Python的基本数据类型大致可分为如下七种：

  • 数字（Number）

  • 字符串（String）

  • 列表（List）

  • 元组（Tuple）

  • 集合（Sets）

  • 字典（Dictionary）

  • 日期（Date）

需要注意的是，Python跟C等语言不太一样，它的变量是不需要声明。但每个变量在使用前必须先赋值，赋值之后变量便会被创建。

例如下面的整数a和字符串str：


a = 2;

b = 'abcd';

附上查看类型的方法：


>>> type(a)

<class 'int'>

>>> type(b)

<class 'str'>

即可打印出变量a和b的数据类型。

好了，下面将对七大数据类型一一进行简单分析。

正文

1 数字 / Number

数字分为：

  • 1 整数 int

  • 2 浮点数 float

  • 3 复数 complex

  • 4 布尔类型 bool

1.1 整数 / int


a = 12;

b = -100;

1.2 浮点数 / float


c = 1.5;

d = -0.15;

1.3 复数 / complex

以下为两种复数的表示方式，其中j为复数单位：


a = 3 + 4j;        # 复数的第一种定义方式

b = complex(2, 3);    # 复数的第二种定义方式

>>> a.real    # 获取a的实部

3

>>> a.imag    # 获取a的虚部

4

>>> abs(a)    # 求a的值

5

1.4 布尔类型 / bool

布尔类型只有两种情况，即真（True）或假（False）：


a = True

b = False

2 字符串 / String

字符串可以使用单引号(‘aaa’)，双引号(‘’aa’’)或三引号(‘’’aaa’’’)来定义括起来。


str1 = 'aaa'

str2 = ''aaa''

str3 = '''aaa'''

若想输出特殊字符，即转义字符反斜杠()：


>>> str4 = 'you don\'t know.'

>>> print( str4 )

you don't know.

3 列表 / List

列表类似于一个清单。里面可以有


list=['google','apple',123];

num=[1,2,3,4,5,6];

>>> print(list[0])

google

>>> print(num[0])

1

>>> print(num[1:3])

[2,3]

4 元组 / Tuple

元组跟列表的区别：

1 元组定义后不能修改；

2 元组用( )，列表用[ ]。


tup1=(1,2,3,4,5);

tup2=('abc','def',123);

tup3=''a'',''b'',''c'';

print tup1[0]

5 集合 / Sets

集合有一个特殊的功能：去重，即去掉重复项。

例如：


list1 = [1, 1, 2, 3]

list1_set = set(list1)

现在list1_set变为了[1, 2, 3]

6 字典 / Dictionary

字典由键值和数值构成，用冒号分开。


dic1={'alexa': 1 , 'bill': 2 , 'cat': 3}

print dic1['alexa']

7 日期 / Date


>>> import datetime

>>> now_time = datetime.datetime.now()

>>> now_time

datetime.datetime(2018,1,1,18,10,7,908836) # 分别对应年 月 日 时 分 秒 毫秒

>>> now_time.year

2018

# 相应的你还可以使用

now_time.month

now_time.day

now_time.hour

now_time.minute

now_time.second

now_time.microsecond

至此，介绍完毕。。
