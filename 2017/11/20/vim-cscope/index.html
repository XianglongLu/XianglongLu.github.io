Linux下的Coder利器 —— cscope的安装,配置及基本指令

前言 / Preface

Linux下的研发，难免会碰到：函数被谁调用，该标签在哪里出现过等问题。
cscope的存在，可以很好的帮我们解决这个问题。

正文 / Body

1. cscope的安装 / How to install cscope

方法1：从官网下载并安装

官网地址为：http://cscope.sourceforge.net/
So you can download the latest version of cscope from the official website.
你则可得到安装包 e.g. cscope-15.8b.tar.gz

解压并安装安装包

$ tar xvf cscope-15.8b.tar.gz
$ cd cscope-15.8b
$ ./configure
$ make        // please ensure it does have any error!
$ sudo make install

确保安装成功

$ which cscope

看到类似于

/usr/local/bin/cscope

的结果，则代表安装成功。

方法2：直接安装

$ sudo apt-get install cscope

则可完成安装。

3. cscope的基本指令

vim下使用cscope命令:
首先需要输入cscope add命令添加一个cscope数据库。
然后就可以使用vim支持的8种cscope命令进行操作了。

:cs find s/g/d/c/t/e/f/i
s: 查找C语言符号，即查找函数名、宏、枚举值等出现的地方
g: 查找函数、宏、枚举等定义的位置，类似ctags所提供的功能
d: 查找本函数调用的函数
c: 查找调用本函数的函数
t: 查找指定的字符串
e: 查找egrep模式，相当于egrep功能，但查找速度快多了
f: 查找并打开文件，类似vim的find功能
i: 查找包含本文件的文件

4. cscope quickfix

将搜索到的结果显示为列表：

:cw

切换上/下一个搜索结果：

:cn \\ 跳至下个搜索结果
:cp \\ 跳至上个搜索结果
:cn2 \\ 跳至下2个搜索结果
:cp3 \\ 跳至上3个搜索结果

5. cscope的实操

下面我使用Linux 4.1.42内核源码来做实操案例。

5.1 生成数据库文件cscope.out

在内核源码路径下输入 

$ cscope -Rb

则会在当前目录下生成数据库文件cscope.out。

5.2 在vim命令模式下添加数据库文件

在当前路径下进入vim命令模式，并添加数据库文件cscope.out：

$ vim
:cs add cscope.out
