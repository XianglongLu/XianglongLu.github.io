Git —— 如何删除远程仓库中的文件或文件夹

前言 / Preface

有时候想删除仓库中的一些文件或文件夹，具体该如何操作呢？方法如下。

正文 / Body

1. 首先把远程仓库clone到本地

$ git clone https://git.coding.net/xxx/xxx.git

2. 删除想要删掉的文件或文件夹

  • 删除文件的指令
    $ git rm xxx_file
  • 删除文件夹的指令
    $ git rm -r xxx_folder

3. 提交代码并添加注释

$ git commit -m "注释"

4. 推送到远程仓库


$ git push origin branch_name

e.g.


$ git push origin master

至此，完成远程仓库中文件或文件的删除。
