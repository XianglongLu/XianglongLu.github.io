Missing /proc/asound directory

最近在调试一款产品的microphone和speaker中，发现/proc目录下并没有熟悉的asound目录。
系统信息如下：

kernel version: 4.4.103
buildroot

查看kernel configure

cd kernel
make menuconfig

选择右下角的，导入所build版本的kernel config file。然后以此进入：

 Device Drivers --->
 Sound card support --->
 Advanced Linux Sound Architecture --->

发现选项：

[] Sound Proc FS Support

并未开启。
将其勾选。

[*] Sound Proc FS Support

重新编译并烧录image。我们熟悉的/proc/asound又回来了。
