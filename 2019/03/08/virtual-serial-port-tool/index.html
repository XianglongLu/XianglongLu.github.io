今天介绍一个用于生成虚拟串口的软件——virtual serial port driver

通过这个软件，你可以在PC上申城一对虚拟串口对。
这对于调试串口通信的同学是十分有帮助~
我用的是试用版，仅有15天的使用时间。


点击右边的Add pair即可生成一对虚拟串口对。


如图所示，我已经生成了2对虚拟串口对。
COM1 <----> COM2
COM2 <----> COM3

现在使用XCOM测试一下COM1和COM2口。

左图为通过COM2向COM1发送”hello！”

右图=图为通过COM1向COM2发送”what？”
