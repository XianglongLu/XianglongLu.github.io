如何通过GPIO sysfs interface控制GPIO

前言 / Preface

正文 / Body

1. 是否生成gpiochipX

请先直接进入sys/class/gpio/目录下，看看是否有gpiochip0，gpiochip1，gpiochipX或类似字样的fs存在。
如果有，请您跳至下一步。
如果没有，通过make menuconfig配置你的内核，具体如下：
-> Device Drivers
-> GPIO Support
-> /sys/class/gpio/… (sysfs interface)
即打开GPIO sysfs interface。
然后重新编译内核并完成烧录。
之后再去sys/class/目录下看看，是否已经生成gpiochipX。

2. 如何通过gpio sysfs控制GPIO

在sys/class/gpio目录下，我们除了可以看到gpiochipX，还会看到export和unexport文件。

我们可以通过

echo 20 > export

生成GPIO 20的节点。
进入该节点中你会发现有：

direction value

之后我们通过echo指令便可实现对GPIO状态的设置和查看。

cat value        // 会返回当前GPIO口是高电平(1)还是低电平(0)
cat direction    // 返回当前GPIO口的输入(in)输出(out)状态
echo 1 > value    // 将GPIO口设为高电平
echo out > direction // 将GPIO口设为输出

后来我发现一个更好的方法，即直接操作pull：

echo up > pull
echo down > pull

即可直接拉高或拉低电平。

3 实例分析

3.1 将GPIO 6设置为输出高电平

$ cd sys/class/gpio/
$ echo 6 > export
$ cd gpio6
$ echo out > direction
$ echo 1 > value

4 C程序例子

#include <stdio.h>
#include <string.h>
#include <stdlib.h>
#include <getopt.h>
#include <sys/types.h>
#include <sys/stat.h>
#include <fcntl.h>

# define GPIO_PULL          "/pull"
# define GPIO_DIRECTION     "/direction"
# define GPIO_VALUE         "/value"
# define GPIO_DIR           "/sys/class/gpio/gpio"

int main()
{
    char string[64];
    int num=10;
    int fd;
    char gpioValue;


// 判断gpio10设备节点是否存在，不存在则建立节点

    snprintf(string, sizeof(string), "/sys/class/gpio/gpio%d", num); 
    if (access(string, 0) == -1) // 0用于判断存在问题
    {
        snprintf(string, sizeof(string), "echo %d > /sys/class/gpio/export", num);
        system(string);
    }    


// 通过打开的方式，读取gpio10节点下value的值。

    snprintf(string, sizeof(string), "/sys/class/gpio/gpio%d/value", num); 

    fd = open(string, O_RDONLY); // 表示仅可读，O_RDWR表示可读可写，O_WRONLY表示仅可写。
    if(fd <= 0) // 无法打开会返回-1
    {
        printf("Open error.\n");
    }
    read(fd, &gpioValue, 1); // 将value里的值赋给gpioValue
    close(fd); // 打开过后记得关闭


// 通过打开的方式，将1写入gpio10节点下value的值。

    gpioValue = 1;
    snprintf(string, sizeof(string), "/sys/class/gpio/gpio%d/value", num); 

    fd = open(string, O_WRONLY);
    if(fd <= 0) // 无法打开会返回-1
    {
        printf("Open error.\n");
    }
    write(fd, &gpioValue, 1); // 将gpioValue的值写入value节点
    close(fd); // 打开过后记得关闭    

    return 0;
}
