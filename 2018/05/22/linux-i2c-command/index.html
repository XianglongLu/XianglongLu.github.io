Linux I2C tools —— i2cdetect, i2cdump, i2cset和i2cget命令解析

Linux kernel 3.1中新添加了I2C tools。
这使得I2C的一些常用命令得到了统一。

I2C tools包括如下4个指令：

i2cdetect
i2cdump
i2cget
i2cset

从名字上就很好理解啦，下面对它们的使用进行一一解析。

i2cdetect

i2cdetect可用于显示当前Linux平台上有几个i2c的bus。

# i2cdetect -l

还可以显示出某个bus下面，都有哪些地址存在设备。
e.g. 看看i2c-1的设备情况

# i2cdetect -y 1

i2cdump

i2cdump用于dump出某个i2c device的register情况。
e.g. 看看i2c-1下0x1a device的register情况

# i2cdump -f -y 1 0x1a

i2cget & i2cset

e.g. 读出i2c-1上address为0x1a的device上register address为0x02处的值

# i2cget -f -y 1 0x1a 0x02
0xc8

命令执行完后会在第二行返回所get的值。

e.g. 将0xc9写入i2c-1上address为0x1a的device上register address为0x02处

# i2cset -f -y 1 0x1a 0x02 0xc9

写入没有返回。
设置完毕后可以通过i2cget命令把数值get回来，看看写入了没有。
