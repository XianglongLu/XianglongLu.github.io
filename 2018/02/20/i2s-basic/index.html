Linux —— I2S音频总线简介

前言

I2S（Inter-IC Sound）总线，即集成电路内置音频总线，是飞利浦公司为数字音频设备之间的音频数据传输而制定的一种总线标准，该总线专责于音频设备之间的数据传输，广泛应用于各种多媒体系统。

I2S一般包含如下3根数据信号线：

1. 位时钟线 / bit clock line（BCLK）
2. 字段时钟线 / word clock line
3. 串行数据线 / serial data line

但有时还会包含：

4. 主时钟 / Master clock（MCLK）
5. A multiplexed data line for upload

1. BCLK / 位时钟

也叫作连续串行时钟（SCLK，continuous serial clock）线。它对应于音频数据的每一位。

频率 = 2 x 采样频率 x 采样位数

2. LRCLK / 左右声道时钟

LRCLK全称left-right clock，也叫作字段选择（WS，word select）线。

该线主要用于切换左右声道的数据。
当WS=0时代表左声道；
WS=1时代表右声道。

频率 = 采样频率

3. SD / 串行数据

SD即serial data。
一般用二进制补码来表示音频数据。

--- 补码知识补充 ---
正数的原码=反码=补码
十进制 +23
二进制原码 00010111
二进制反码 00010111
二进制补码 00010111

负数的补码=反码+1
十进制 -23
二进制原码 10010111
二进制反码 11101000 // 符号位不变，其它位取反
二进制补码 11101001 // 反码+1

4. MCLK / 主时钟

有时为了使系统间能够更好地同步，还需要另外传输主时钟（Master Clock, MCLK），也叫系统时钟（System Clock）。
